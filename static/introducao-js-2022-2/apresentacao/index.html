Javascript?

Utilizado em diferentes plataformas

  • Navegadores
  • Server side script
    • Node.js
  • Internet of things (IoT)
    • Espruino
    • Tessel
    • Johnny five framework

Javascript?

  • Não é Java
    • Sintaxe parecida
    • Tipagem fraca
    • Sem tipos estáticos
  • Também conhecido como ECMAScript
  • Multiplataforma
  • Multiparadigma
    • Funções de primeira classe
      • Função é um tipo linguagem
      • Orientado a objetos
        • "Prototype-based", ao invés de "class based"1
      • Funcional
      • Imperativo
1 - A versão 6 do ECMAScript adiciona suporte a classes

    • Parte do Internet Explorer 3

Variáveis

  • Nomes de variáveis podem iniciar com os caracteres de A até Z (e também a até z), _ (underscore) ou $ (dólar)
  • Podem ter diferentes escopos léxicos (dependerá de como é definida)
    • var tem escopo de função
    • const e let tem escopo de bloco

Herança ("Old School")


function Animal(nome) {
  this.nome = nome;
}
Animal.prototype.comer = function() {
  console.log(this.nome, " está comendo");
};
Animal.prototype.beber = function() {
  console.log(this.nome, " está bebendo");
};
function Cao(nome) {
  Animal.call(this, nome);
}
Cao.prototype = new Animal;
Cao.prototype.constructor = Cao;
Cao.prototype.latir = function() {
  console.log(this.nome, " está latindo");
};
var cao = new Cao("Maisena");
cao.comer();
cao.beber();
cao.latir();
